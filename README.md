  ## ⚠️ FOR TERMUX/UBUNTU/SSH USER

```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
apt install ffmpeg -y
git clone https://github.com/DarkWinzo/Queen-MD
cd Queen-MD
npm install
npm i pm2
pm2 start index.js && pm2 save && pm2 logs
```
